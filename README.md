- `read_data.R`:  R code to read TCRseq data and save to RData for following process
- `process.R`: calculate classifier result for each sample with different threshold
  - `Rscript process_test.R $compare $p $input`
    - `$compare`: the comparison group, e.g. "c.vs.p" as control vs. previvor
    - `$p`: sample name
    - `$input`: the input RDS file generated by `read_data.R`
- `summarize.R`: summarize result from `process.R` and generate ROC curve
- `Manuscript_code.R`: code to generate figures in the manuscript
